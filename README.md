# Maze / Pathfinding Visualizer

A simple grid-based pathfinding visualizer written in HTML, CSS, and vanilla JavaScript.

## Features

- Click to place/remove walls
- Set **Start (S)** and **End (E)** positions
- Visualize **Breadth-First Search (BFS)** step by step

## Controls

- Left click: toggle wall (when in "wall" mode)
- Press **S**: switch to "set start" mode, then click a cell
- Press **E**: switch to "set end" mode, then click a cell
- Press **W**: switch back to "wall" mode
- **Start Search** button: runs the chosen algorithm
- **Clear Grid**: clears walls & visited/path cells

## Running

Simply open `index.html` in a browser.
